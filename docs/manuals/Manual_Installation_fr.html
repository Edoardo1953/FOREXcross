Guide d'Installation

Ajouter à l'écran d'accueil

Transformez la Web App en une application native sur votre mobile.

Pour les Appareils Android (Chrome)

1
Ouvrez le site dans Chrome Naviguez vers l'adresse de notre application en utilisant le navigateur Google Chrome sur votre smartphone.
2
Ouvrez le menu Options Appuyez sur l'icône avec les trois points (⋮) dans le coin supérieur droit de la barre Chrome.
3
Installez l'application Sélectionnez "Installer l'application" ou "Ajouter à l'écran d'accueil".
4
Confirmez Appuyez sur "Ajouter" dans la fenêtre qui apparaît. L'icône de l'application apparaîtra instantanément sur votre écran d'accueil avec vos autres applications.

Pour les Appareils iOS / iPhone (Safari)

1
Ouvrez le site dans Safari Il est essentiel d'utiliser Safari sur iPhone pour pouvoir installer l'application correctement.
2
Appuyez sur l'icône Partager Appuyez sur le bouton Partager (le carré avec une flèche vers le haut) situé dans la barre inférieure.
3
Ajouter à l'écran d'accueil Faites défiler la liste des options vers le bas et appuyez sur "Sur l'écran d'accueil".
4
Terminez l'opération Appuyez sur "Ajouter" dans le coin supérieur droit. L'icône FOREXcross apparaîtra sur l'écran d'accueil de votre iPhone.
Pourquoi l'installer ? Une fois ajoutée à l'écran d'accueil, l'application s'ouvrira sans les barres du navigateur, vous offrant une expérience plein écran plus rapide, plus fluide et plus professionnelle, tout comme une application téléchargée sur les Stores officiels.
