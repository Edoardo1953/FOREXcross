FOREXcross User Manual

Comprehensive Guide to the Currency Exchange Dashboard

1. General Overview

FOREXcross is an advanced dashboard designed for real-time monitoring and historical analysis of exchange rates. The application provides a clear view of global currency markets, with specific tools for calculating cross rates and consulting historical databases.

2. Main Features

Exchange Rates Dashboard

The main screen shows the current exchange rates referred to the Euro (EUR) or US Dollar (USD). Data is updated periodically to ensure accuracy.

Main Crosses

Quick access to the most traded currency pairs (EUR/USD, GBP/USD, USD/JPY). By clicking on the desired pair, the entire dashboard updates to show the relative data.

Custom Cross (Personalized Calculation)

A powerful tool to calculate the exchange between any pair of currencies not present in the quick menus. Select the source currency, the target currency, and instantly get the conversion rate.

3. Historical Analysis and Database

The app allows you to travel back in time to consult currency trends:

  • Historical Rates: View closing rates from past dates.
  • Historical Database: Access to a complete archive for extended comparative analysis.
  • Interactive Charts: Variations are graphically represented for immediate understanding of market trends.

4. Multi-Device Experience

FOREXcross is optimized for every screen:

  • PC & Tablet: Full view with side menu and expanded charts.
  • Mobile (CELL): Compact interface with a collapsible menu. You can test the mobile view even from a PC using the "CELL" button in the header.

5. Tools and Privacy

Through the gear icon (Settings) or key icon (Privacy), you can:

  • Manuals: Consult and download this guide and the privacy documentation.
  • Clear Cache: Clear temporary data to speed up the application.
  • Access Sharing: Generate password-protected links with expiration times to allow collaborators to view the dashboard securely.
💡 Pro Tip: Use the "Download PDF" function within the manuals to keep a local copy of the documents on your device, accessible even without an internet connection.
